Android application for connecting to the Glucosense non-invasive glucose prototype via Bluetooth.
The app acts as a transparent client that receives processed data from the ESP32-based device and displays signal status, features, and glucose trend estimation in real time.
Research prototype only. Not a medical device.
Overview
The Glucosense Android app is designed to:
Connect to ESP32 via Bluetooth
Receive measurement data as structured strings
Parse and display optical features
Show measurement state (calibration / ready / warning)
Display glucose trend prediction
Provide raw data monitoring mode
The app focuses on clarity, transparency, and real-time feedback during measurement.
